J. M., Fedorov, A., Parmar, C., Hosny, A., Aucoin, N., Narayan, V., Beets-Tan, R. G. H., Fillon-Robin, J. C., Pieper, S., Aerts, H. J. W. L. (2017). Computational Radiomics System to Decode the Radiographic Phenotype. Cancer Research, 77(21), e104–e107.</p></li></ul><h3>1.1.2 主要内容</h3><p><code>PyRadiomics</code>可以针对医学影像数据(如CT、PET、MRI)中提取影像组学数据进行图像和分割图的加载和预处理、过滤器应用、特征计算并返回结果(Fig.1A)。</p><ul><li><p>Figure 1<img data-imgfileid="100021695" data-ratio="1.11125" data-src="https://mmbiz.qpic.cn/mmbiz_png/ImlBFVOwwpx2dVa7tC0UYQRDUr6TnenAc4ZAfEX54Nsmic2g7JMtXNzewUvVc5mp8HPU8Mn14E7hyas9gzxJt7w/640?wx_fmt=png&amp;from=appmsg" data-type="png" data-w="800" src="https://mmbiz.qpic.cn/mmbiz_png/ImlBFVOwwpx2dVa7tC0UYQRDUr6TnenAc4ZAfEX54Nsmic2g7JMtXNzewUvVc5mp8HPU8Mn14E7hyas9gzxJt7w/640?wx_fmt=png&amp;from=appmsg"></p></li><li><p>1.数据加载和预处理<br>绝大多数图像处理都是使用<code>SimpleITK</code>完成的，其开源工具<code>Insight Toolkit</code>（<code>ITK</code>） 提供了简化的界面。这使得<code>PyRadiomics</code>能够支持各种图像格式，同时也确保许多低级功能和基本图像处理都经过彻底的测试和维护。对于纹理和形状特征，包括多个重采样选项，以确保各向同性体素在所有方向上相邻体素之间的距离相等。</p></li><li><p>2.过滤<br>可以在原始图像上计算特征，也可以使用多个内置过滤器选择在预处理的图像上计算特征。其中包括<code>wavelet</code>和<code>laplacian of gaussian</code>（<code>LoG</code>） 滤波器，以及几个简单的过滤器，包括<code>square</code>, <code>square root</code>, <code>logarithm</code>和<code>exponential</code> 。对于<code>wavelet</code>和 <code>LoG</code> 过滤器的应用，<code>PyRadiomics</code>分别使用了<code>PyWavelets</code>和<code>SimpleITK</code>。其余过滤器是基于<code>NumPy</code>实现的。</p></li><li><p>3.特征计算<br>特征的计算主要包含五个<code>feature</code>的类：<code>first-order statistics</code>, <code>shape descriptors</code>, <code>texture classes Gray Level Co-occurrence Matrix</code>, <code>Gray Level Run Length Matrix</code>, <code>Gray Level Size Zone Matrix</code>,除了<code>shape descriptors</code>外均可由过滤或未过滤过的图像生成，<code>shape descriptors</code>只能通过未过滤图像生成<code>shape descriptors</code>。特征计算直至单个的2D图像以及3D图像</p></li><li><p>4.结果 计算出的特征在有序<code>Python</code>字典中存储和返回。每个要素都由一个唯一名称标识，该名称由应用的过滤器、要素类和要素名称组成。除了计算的特征外，该字典还包含有关提取的其他信息，包括当前版本、应用的过滤器、设置和原始图像间距。为了增强可用性，PyRadiomics 有一个模块化的实现，以特征提取器模块为中心，该模块定义了特征提取管道并处理与平台中其他模块的交互。所有要素类都在单独的模块中定义。此外，所有这些都继承自基本特征提取类，从而提供了一个通用接口。最后，该平台包含两个帮助程序模块，一个是<code>generalinfo</code>，前者提供返回结果中包含的其他提取信息，另一个是<code>imageoperations</code>模块，前者用于实现图像预处理和过滤器期间使用的功能。除了通过 <code>featureextractor</code>模块在<code>Python</code>脚本中交互式使用外，<code>PyRadiomics</code>还支持从命令行直接使用。有两个脚本可用：<code>pyradiomics</code>和<code>pyradiomicsbatch</code>，分别用于单图像和批处理。对于这两个脚本，可以使用额外的参数文件来自定义提取，并且可以将结果直接导入到许多统计的<code>Python</code>包中进行分析，甚至可用于<code>R</code>和<code>SPSS</code>的分析。</p></li><li><p>5.示例数据<br>在案例研究中，作者展示了<code>PyRadiomics</code>在肺部病变表征中的应用，以区分<strong>良性</strong>和<strong>恶性</strong>结节。数据为肺图像数据库联盟（LIDC-IDRI）的公开队列，该队列包括诊断和肺癌筛查CT扫描，以及来自经验丰富的放射科医生的标记注释病灶和每个病灶的恶性肿瘤评级。在302名患者中，作者在分析中纳入了429个不同的病变，每个病变都有四个体积分割和恶性肿瘤评级。总共从每个病变的所有四个描绘中提取了 1120 个影像组学特征（14 个<code>shape features</code>、19 个<code>first-order intensity statistics features</code>、60 个<code>texture features</code>、395 个<code>LOG features</code>和 632 个<code>wavelet features</code>）。</p></li></ul><p>为了评估手动分割对影像组学特征值的影响变化，作者计算了从放射科专家进行的四次分割中提取的每个特征的稳定性。该稳定性是使用类内相关系数 （ICC） （Fig.1B).LOG±（ICC= 0.8 ± 0.91）、一阶强度统计（ICC= 0.11 ± 0.88）和纹理特征（ICC= 0.13 ± 0.91）表现出较高的稳定性（中值 sd：ICC ±&gt; 0.11）和小波（ICC= 0.60 ± 0.31）特征表现出中等稳定性，表明它们对描绘变异性敏感。</p><p>选择所有具有高稳定性的特征（ICC&gt;0.8），得到535个放射组学特征（5个形状特征，14个<code>first-order intensity statistics features</code>，48个<code>texture features</code>，310个<code>LOG feature</code>和158个<code>wavelet features</code>）。图1C显示 535 个结节（列）中 429 个稳定放射组学特征（行）的标准化表达值的无监督聚类。作者观察到四个不同的病变簇，具有相似的表达值。将这些集群与病变恶性肿瘤状态进行比较，作者观察到它们之间存在显着差异（P = 2.56e−24）。92%（n= 81） 的S1 簇（n = 88） 样本为恶性，而 95% （n = 38） 的 S2 簇 （n = 40） 样本为良性。对于S3簇（n = 143） 和 S4 （n = 158），恶性样本的比例分别为 54% （n = 78） 和 34% （ n = 53）。这些结果证明了基于影像学的亚型与肺部病变恶性状态之间的关联。为了评估多变量成像生物标志物的性能，将队列分为训练组（n=214）和验证组（n=215）。使用最小冗余最大相关性 （mRMR），作者从训练队列中选择了 25 个稳定的影像组学特征。基于训练数据，通过将选定的特征拟合到随机森林分类器中，开发了一种多变量生物标志物。该生物标志物在表征肺结节方面表现出强大而显着的性能（AUC=0.79）（Figure 1D)。</p><h2>1.2
