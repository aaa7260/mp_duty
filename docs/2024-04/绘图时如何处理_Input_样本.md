<strong>ChIP/cut&amp;tag/MeRIP seq</strong> 等之类的数据时,往往需要对应的 <strong>Input/IgG</strong> 样本作为对照,来去除背景噪音,鉴定出准确的结合区域(<strong>peak</strong>)。在后续的绘图中, 是否需要将 <strong>Input</strong>
