data-tool="mdnice编辑器">平常在撰写论文时，会需要比较算法之间的计算时间。本篇文章给出几种测量 R 代码运行时间的方法。本文是小编学习过程中的笔记，主要参考博客<span>1</span><sup>[1]</sup>，<span>2</span><sup>[2]</sup>。</p><h2 data-tool="mdnice编辑器"><span></span><span>1. 使用 <code>Sys.time()</code></span></h2><p data-tool="mdnice编辑器">小编通常使用 <code>Sys.time()</code> 函数来计算时间。首先记录当前运行时刻，之后运行代码，并记录运行完成时刻的时间。最后将两者时间相减即可。下面给出一个例子：</p><pre data-tool="mdnice编辑器"><span></span><code>myfunction &lt;- <span>function</span>() { <br>Sys.sleep(<span>10</span>) <span>#系统休息 </span><br>}<br><br>start_time &lt;- Sys.time() <span># 记录初始时间</span><br>myfunction() <span># 运行你的代码</span><br>end_time &lt;- Sys.time() <span># 记录终止时间</span><br><br>end_time - start_time <span># 计算时间差</span><br><span># Time difference of 10.00363 secs</span><br></code></pre><h2 data-tool="mdnice编辑器"><span></span><span>2. 使用 toctoc 包</span></h2><p data-tool="mdnice编辑器">函数 <code>tic()</code> 和 <code>toc()</code> 的使用方式与前面演示的 <code>Sys.time()</code> 类似。但是 <span>toctoc</span><sup>[3]</sup> 包更加方便。下面给出例子：</p><p data-tool="mdnice编辑器">读者可以从 <span>GitHub</span><sup>[4]</sup> 或者 <span>CRAN</span><sup>[5]</sup> 上安装：</p><pre data-tool="mdnice编辑器"><span></span><code><span># CRAN</span><br>install.packages(<span>"tictoc"</span>)<br><span># GitHub</span><br><span>library</span>(devtools); devtools::install_github(<span>"collectivemedia/tictoc"</span>)<br></code></pre><p data-tool="mdnice编辑器">计算单个代码块</p><pre data-tool="mdnice编辑器"><span></span><code><span>library</span>(tictoc)<br>tic(<span>"sleeping"</span>)<br>print(<span>"falling asleep..."</span>)<br>sleep_for_a_minute()<br>print(<span>"...waking up"</span>)<br>toc()<br><span># [1] "falling asleep..."</span><br><span># [1] "...waking up"</span><br><span># sleeping: 11.076 sec elapsed</span><br></code></pre><p data-tool="mdnice编辑器">嵌套多个计时器，注意这里的第一个 <code>tic()</code> 对应最后一个 <code>toc()</code>。</p><pre data-tool="mdnice编辑器"><span></span><code>tic(<span>"total"</span>)  <span>#对应最后一个toc()</span><br>tic(<span>"data generation"</span>)<br>X &lt;- matrix(rnorm(<span>50000</span>*<span>1000</span>), <span>50000</span>, <span>1000</span>)<br>b &lt;- sample(<span>1</span>:<span>1000</span>, <span>1000</span>)<br>y &lt;- runif(<span>1</span>) + X %*% b + rnorm(<span>50000</span>)<br>toc()<br>tic(<span>"model fitting"</span>)<br>model &lt;- lm(y ~ X)<br>toc()<br>toc()<br><span># data generation: 3.383 sec elapsed</span><br><span># model fitting: 42.452 sec elapsed</span><br><span># total: 46.428 sec elapsed</span><br></code></pre><h2 data-tool="mdnice编辑器"><span></span><span>3. 使用 <code>system.time()</code></span></h2><p data-tool="mdnice编辑器">这个使用方式类似 <code>Sys.time()</code>，我们继续以 <code>myfunction()</code> 作为例子：</p><pre data-tool="mdnice编辑器"><span></span><code>system.time({ myfunction() })<br><span>#  user    system  elapsed </span><br><span>#  0.039   0.044   10.001</span><br></code></pre><h2 data-tool="mdnice编辑器"><span></span><span>4. 使用 <code>rbenchmark</code> 包</span></h2><p data-tool="mdnice编辑器">该包中的 <code>microbenchmark()</code> 函数可以进行精确的测量和比较 R 表达式的执行时间。它提供了更准确的结果，可以替代 <code>system.time()</code> 使用。</p><p data-tool="mdnice编辑器">可以通过 <span>CRAN</span><sup>[6]</sup> 或者 <span>GitHub</span><sup>[7]</sup> 安装：</p><pre data-tool="mdnice编辑器"><span></span><code><span># CRAN</span><br>install.packages(<span>"rbenchmark"</span>)<br><span># GitHub</span><br>devtools::install_github(<span>"eddelbuettel/rbenchmark"</span>)<br></code></pre><p data-tool="mdnice编辑器">这里展示<span>R-bloggers</span><sup>[8]</sup>中的例子：使用三个计算方法比较计算线性回归系数所需的时间：</p><pre data-tool="mdnice编辑器"><span></span><code><span>library</span>(rbenchmark)<br><br>benchmark(<span>"lm"</span> = {<br>            X &lt;- matrix(rnorm(<span>1000</span>), <span>100</span>, <span>10</span>)<br>            y &lt;- X %*% sample(<span>1</span>:<span>10</span>, <span>10</span>) + rnorm(<span>100</span>)<br>            b &lt;- lm(y ~ X + <span>0</span>)$coef<br>          },<br>          <span>"pseudoinverse"</span> = {<br>            X &lt;- matrix(rnorm(<span>1000</span>), <span>100</span>, <span>10</span>)<br>            y &lt;- X %*% sample(<span>1</span>:<span>10</span>, <span>10</span>) + rnorm(<span>100</span>)<br>            b &lt;- solve(t(X) %*% X) %*% t(X) %*% y<br>          },<br>          <span>"linear system"</span> = {<br>            X &lt;- matrix(rnorm(<span>1000</span>), <span>100</span>, <span>10</span>)<br>            y &lt;- X %*% sample(<span>1</span>:<span>10</span>, <span>10</span>) + rnorm(<span>100</span>)<br>            b &lt;- solve(t(X) %*% X, t(X) %*% y)<br>          },<br>          replications = <span>1000</span>,<br>          columns = c(<span>"test"</span>, <span>"replications"</span>, <span>"elapsed"</span>,<br>                      <span>"relative"</span>, <span>"user.self"</span>, <span>"sys.self"</span>))<br><br><span>#           test   replications elapsed relative user.self sys.self</span><br><span># 3 linear system         1000   0.064    1.000     0.062    0.001</span><br><span># 1            lm         1000   0.281    4.391     0.267    0.011</span><br><span># 2 pseudoinverse         1000   0.082    1.281     0.079    0.002</span><br></code></pre><p data-tool="mdnice编辑器">观察 elapsed 的结果，可以看到 <code>lm</code> 方法计算时间是最长的。</p><blockquote data-tool="mdnice编辑器"><p><strong>注意</strong>：读者想使用该包进行计算时间，需要按照文章的形式，将自己的代码填入即可。</p></blockquote><h2 data-tool="mdnice编辑器"><span></span><span>5. <code>microbenchmark</code> 包</span></h2><p data-tool="mdnice编辑器">该包是 <code>rbenchmark</code> 包的升级版本，不仅可以给出上面的结果，还能通过可视化的方式展示结果。</p><p data-tool="mdnice编辑器">可以通过 <span>CRAN</span><sup>[9]</sup> 或者 <span>GitHub</span><sup>[10]</sup> 安装：</p><pre data-tool="mdnice编辑器"><span></span><code><span># CRAN</span><br>install.packages(<span>"microbenchmark"</span>)<br><span># GitHub</span><br>remotes::install_github(<span>"joshuaulrich/microbenchmark"</span>)<br></code></pre><p data-tool="mdnice编辑器">该函数能够使用用户指定函数自动检查基准表达式的结果。我们再次比较三种计算线性模型系数向量的方法（重复运行了100次）。</p><pre data-tool="mdnice编辑器"><span></span><code><span>library</span>(microbenchmark)<br><br>set.seed(<span>2017</span>)<br>n &lt;- <span>10000</span><br>p &lt;- <span>100</span><br>X &lt;- matrix(rnorm(n*p), n, p)<br>y &lt;- X %*% rnorm(p) + rnorm(<span>100</span>)<br><br>check_for_equal_coefs &lt;- <span>function</span>(values) {<br>  tol &lt;- <span>1e-12</span>  <span># 添加基准</span><br>  max_error &lt;- max(c(abs(values[[<span>1</span>]] - values[[<span>2</span>]]),<br>                     abs(values[[<span>2</span>]] - values[[<span>3</span>]]),<br>                     abs(values[[<span>1</span>]] - values[[<span>3</span>]])))<br>  max_error &lt; tol<br>}<br><br>mbm &lt;- microbenchmark(<span>"lm"</span> = { b &lt;- lm(y ~ X + <span>0</span>)$coef },<br>               <span>"pseudoinverse"</span> = {<br>                 b &lt;- solve(t(X) %*% X) %*% t(X) %*% y<br>               },<br>               <span>"linear system"</span> = {<br>                 b &lt;- solve(t(X) %*% X, t(X) %*% y)<br>               },<br>               times = <span>100</span>, <span>#重复运行了100次</span><br>               check = check_for_equal_coefs) <span># 添加自通检查的标准</span><br><br>mbm<br><span># Unit: milliseconds</span><br><span>#           expr      min        lq      mean    median        uq      max neval cld</span><br><span>#             lm 96.12717 124.43298 150.72674 135.12729 188.32154 236.4910   100   c</span><br><span>#  pseudoinverse 26.61816  28.81151  53.32246  30.69587  80.61303 145.0489   100  b</span><br><span>#  linear system 16.70331  18.58778  35.14599  19.48467  22.69537 138.6660   100 a</span><br></code></pre><blockquote data-tool="mdnice编辑器"><p><strong>注意</strong>：我们使用函数参数检查来检查三种方法返回的结果的相等性（最大误差为<code>1e-12</code>）。如果结果不相等，则 <code>microbenchmark</code> 将返回错误消息。</p></blockquote><p data-tool="mdnice编辑器">值得一提的是，该包还整合了 <code>ggplot2</code> 来绘制以上结果。你只需要使用 <code>autoplot()</code> 即可实现：</p><pre data-tool="mdnice编辑器"><span></span><code><span>library</span>(ggplot2)<br>autoplot(mbm)<br></code></pre><figure
