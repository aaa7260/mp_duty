powered-by="xiumi.us"><p><span></span>大家好！为大家介绍一篇 2022 年发表在<em><strong>Nature Communications</strong></em>的文章，题目为 “<strong>De novo analysis of bulk RNA-seq data at spatially resolved single-cell resolution</strong>” 。主要介绍了一种基于深度学习框架的空间反卷积算法Bulk2Space，以现有的高质量单细胞和空间转录组数据为参考，从bulk RNA-seq数据中生成单细胞分辨率的空间表达谱。通讯作者是来自于浙江大学药学院的范骁辉教授、浙江大学计算机科学与技术学院的陈华钧教授以及军事医学科学院高月教授。<span></span></p></section><section powered-by="xiumi.us"><section><svg viewbox="0 0 1 1"></svg></section></section></section></section></section></section><section powered-by="xiumi.us"><section><span>组织的复杂性体现在细胞异质性和空间分布多样性。空间转录组学的发展使得在空间水平上了解细胞组成、分子结构以及组织功能成为可能。然而目前的空间转录组测序技术还无法实现单细胞分辨率，并且过程耗时、成本高，难以大规模应用。同时，bulk RNA-seq已被广泛应用于转录组分析，过去二十多年间开展了许多大型项目，如ENCODE、TCGA和ICGC，留下大量bulk RNA-seq数据。重新分析已有bulk数据，挖掘其中的细胞多样性和空间表达模式是一项具有挑战性且意义重大的任务。<br></span></section><section><span>计算分析方法具有从bulk RNA-seq数据推测单细胞分辨率下空间异质性的潜力，然而已发表的CPM、CIBERSORT和MuSiC等方法只能从bulk RNA-seq数据中推断各细胞类型的比例，无法进一步生成单细胞数据并映射单细胞到组织空间坐标。因此，需要一种能够将bulk RNA-seq数据有效解卷积为单细胞空间表达谱的算法，能够同时揭示复杂组织的细胞多样性和空间表达变化。</span></section></section><section powered-by="xiumi.us"><section><section powered-by="xiumi.us"><section><section powered-by="xiumi.us"><section><section><svg viewbox="0 0 1 1"></svg></section></section></section></section><section><section powered-by="xiumi.us"><section><p><strong>设计思路</strong></p></section></section></section><section><section powered-by="xiumi.us"><section><svg viewbox="0 0 1 1"></svg></section></section></section></section></section></section><section powered-by="xiumi.us"><p><span><span>文章开发了一种基于深度学习框架的空间反卷积算法Bulk2Space，使用现有的高质量scRNA-seq数据和空间转录组数据作为参考，从bulk转录组中生成单细胞空间表达谱。算法的总体设计如图1所示，分为反卷积和空间映射两个步骤。反卷积使用scRNA-seq数据为参考，假设bulk转录表达等于各细胞类型的基因表达与占比的乘积，通过求解非线性方程确定各细胞类型的占比。接着，利用深层生成模型β-VAE，在每种细胞类型的特征聚类空间中模拟给定数量的单细胞，从而将bulk数据解卷积为单细胞转录组数据（图1b）。在空间映射步骤，作者考虑以两种最常用的空间转录组测序数据为参考，即基于空间编码的RNA-seq方法和基于靶向成像的方法。对于空间编码的数据，算法首先计算每个编码点的细胞类型组成，然后根据表达谱相似度将解卷积得到的单细胞映射到编码点上，同时保证每个点内的细胞类型组成与计算所得一致（图1c）。而对于靶向成像的数据，算法则是根据细胞间共有基因的表达相似性，将解卷积得到的单细胞映射到参考空间的细胞优化坐标上（图1d）。由此产生包含空间位置信息的单细胞RNA-seq数据，可以提供单个细胞的无偏转录组，提高基因覆盖率。</span></span></p><section><img powered-by="xiumi.us"><section><p><strong>总结</strong></p></section></section></section><section><section powered-by="xiumi.us"><section><svg viewbox="0 0 1 1"></svg></section></section></section></section></section></section><section powered-by="xiumi.us"><p><span>     </span><span><span>文章开发了一种基于深度学习模型</span><span lang="EN-US">β-VAE</span><span>的</span><span lang="EN-US">bulk</span><span>空间解卷积算法</span></span>Bulk2Space<span><span>，以现有高质量单细胞转录组数据和空间转录组数据为参考，在每个细胞类型的聚类空间内生成具有生物学意义的单细胞表达谱，并将单细胞精确地映射到组织坐标中，从而将</span><span lang="EN-US">bulk</span><span>转录组数据解卷积为单细胞分辨率的空间转录组数据。对于基于成像的原位杂交技术，</span><span lang="EN-US">Bulk2Space</span><span>可以预测靶基因以外的</span><span lang="EN-US">mRNA</span><span>空间表达。对于空间编码的</span><span lang="EN-US">Bulk2Space</span><span>具有很大的生物学和临床应用前景，有利于从单细胞空间分辨率和全基因覆盖的角度理解组织内的转录异质性，将组织分子特征与表型相关联，揭示细胞的空间特异性变异，发现新基因的空间表达模式，实现更精细的细胞聚类，以及预测疾病进展的分子机制等。</span></span></p></section><section
