lymphocytes (B cells) 和 T lymphocytes (T cells)。B cells 和 T cells 的区别在于表达的 antigen receptor 的结构（structure），分别是B-cell receptor (BCR) 和 T-cell receptor (TCR)。<img data-ratio="0.5258467023172906" data-src="https://mmbiz.qpic.cn/mmbiz_png/iaRJcrq2Los9m59HbictrPva9254uHN8auvw6cXR69YUaWKrbb1dToDHedOCHBlERTDBPhS8MswFy2pzHQnLChCA/640?wx_fmt=png&amp;wxfrom=5&amp;wx_lazy=1&amp;wx_co=1" data-type="png" data-w="1122" src="https://mmbiz.qpic.cn/mmbiz_png/iaRJcrq2Los9m59HbictrPva9254uHN8auvw6cXR69YUaWKrbb1dToDHedOCHBlERTDBPhS8MswFy2pzHQnLChCA/640?wx_fmt=png&amp;wxfrom=5&amp;wx_lazy=1&amp;wx_co=1"></p><p>BCR：由两条重链（IgH）和两条轻链(Igκ，Igλ)组成。</p><p>TCR：由α、β两条肽链组成。BCR的重链包含1个可变区域（VH）和3个恒定区域（CH1/CH2/CH3），轻链包含1个可变区域（VL）和1个恒定区域（CL）。TCR的α、β两条肽链也包含可变区V和恒定区C。</p><p>可变区域（VH/VL）由三个互补决定区（complementarities determining region, CDR） CDR1、CDR2、 CDR3 和间隔的 4 个骨架区（framework region, FR）构成 。CDR1、 CDR2 和 FR 区域相对保守， CDR3 变异最大，并直接决定了 BCR/TCR 的抗原结合特异性。BCR/TCR 的 CDR3 由 V、 D、 J 三个基因编码，淋巴细胞的成熟过程中，通过 V、 D、 J 基因的重排形成了各种重组序列片段，由于 V、 D、 J 基因片段本身具有多样性，再加上体细胞突变、 V（D） J 区之间碱基的插入删除（Indel）等形成了 T/B 细胞的多样性。因此对于淋巴细胞多样性的研究便集中在了 CDR3 区多样性的研究。<img data-ratio="0.2607636068237206" data-src="https://mmbiz.qpic.cn/mmbiz_png/iaRJcrq2Los9m59HbictrPva9254uHN8auia9PTiaBsRVXEgRbm4Gw8UbG9wACepia0vczzdVYLK1OWEKLWWCDOicFRQ/640?wx_fmt=png&amp;wxfrom=5&amp;wx_lazy=1&amp;wx_co=1" data-type="png" data-w="1231" src="https://mmbiz.qpic.cn/mmbiz_png/iaRJcrq2Los9m59HbictrPva9254uHN8auia9PTiaBsRVXEgRbm4Gw8UbG9wACepia0vczzdVYLK1OWEKLWWCDOicFRQ/640?wx_fmt=png&amp;wxfrom=5&amp;wx_lazy=1&amp;wx_co=1"></p><h2><span>基础知识理解</span><span></span></h2><p><strong>免疫组库</strong>：V(D)J序列多样性的集合即为免疫组库（Immune Repertoire, IR）</p><p><strong>免疫组库测序目标</strong>：5‘转录组+V(D)J片段分析，可同时获得单细胞转录组以及免疫组库数据。（V(D)J区段在mRNA的5‘端）</p><p><strong>V(D)J的区段的长度</strong>：通常是650bp左右。PE-150测序最多能测300bp。</p><p><strong>V(D)J 基因的 Reads 比对</strong>：Reads 富集前，先使用 cutadapt 对 Read-pairs 去接头和引物序列。随后将 Read-pairs 比对到 V(D)J 基因片段上，只有比对上的 read 用于后续的组装。其中 read 比对要求：比对得分超过 20；至少有 15bp 能够精确匹配到参考序列。</p><p><strong>Contig</strong>：一致性序列组装得到的contigs序列。组装的过程中，每个 barcode 是独立进行分析的。对过滤后的 read 按照 barcode 分组，其中每个 barcode 至多有 100k 的 reads 用于组装，避免人为因素影响导致的极高覆盖度情况。且仅使用有多于 10 个 reads 的 UMI 的 read 用于组装，最后获得 contigs 序列。</p><p><strong>组装后的有效细胞数</strong>：有效的细胞数的估计与表达靶向 V(D)J 转录本相关，要求 barcode 有一个 contig 能够有至少 2 个具有足够 reads 支持的 UMI。</p><p><strong>contigs注释</strong>：使用 Smith-Waterman 将每个细胞中每个组装的 congtigs 与所有的参考序列（V ， D， J， C 和 5'UTR ）逐一进行比对。</p><p><strong>contigs有效性评价</strong>：</p><ol><li><section>完全跨越 V 和 J 部分，否则为无效 “None”；</section></li><li><section>在预定的 V 位置包含可识别的起始密码子。如果没有找到起始密码子，则认为是无效的 “False”；</section></li><li><section>在 V 起始密码子框内包含一个 CDR3 区域。如果没有发现 CDR3，则被认为是无效的；</section></li><li><section>在 V - J 跨越区不含终止密码子。如果发现终止密码子，则被认为是无效的。理想情况下，每个 Barcode 包含两个匹配的有效的 contigs，由 1 个 TRA 和 1 个 TRB 或 1 个重链（ IGH）和 1 个轻链（ IGK 或 IGL）构成。<img data-ratio="0.41818181818181815" data-src="https://mmbiz.qpic.cn/mmbiz_png/iaRJcrq2Los9m59HbictrPva9254uHN8auW6O4AvG7BiclAPu5yEddjdTE606pE4rkls0iaVC6OlwERvnmJDBfkspQ/640?wx_fmt=png&amp;wxfrom=5&amp;wx_lazy=1&amp;wx_co=1" data-type="png" data-w="1155" src="https://mmbiz.qpic.cn/mmbiz_png/iaRJcrq2Los9m59HbictrPva9254uHN8auW6O4AvG7BiclAPu5yEddjdTE606pE4rkls0iaVC6OlwERvnmJDBfkspQ/640?wx_fmt=png&amp;wxfrom=5&amp;wx_lazy=1&amp;wx_co=1"></section></li></ol><p><strong>Clonotypes克隆型</strong>：通过精确的核苷酸匹配，把拥有完全相同的有效 CDR3 核酸序列的 Cell barcodes 组合在一起， 成为克隆型。</p><p><strong>clonotype的丰度</strong>：细胞支持数（barcode个数）即为clonotype丰度。</p><p><strong>consensus 序列</strong>：所有细胞中拥有相同的有效 CDR3 核酸序列的 contig 组装在一起的最终序列，被称为共有序列。即把某个 clonotype 的所有 contig 组装在一起得到的序列成为 consensus 序列</p><p><strong>克隆型的多样性</strong>：克隆型中的 V-J 基因组合方式的多样性就是克隆型的多样性。</p><h2><span>cellranger mkvdjref</span><span></span></h2><p>VDJ的参考基因组需要使用 cellranger
