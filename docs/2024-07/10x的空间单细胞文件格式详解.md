data-tool="mdnice编辑器"><span></span>首先是表达量矩阵<span></span></h3><p data-tool="mdnice编辑器">目前在单细胞转录组学中，表达量矩阵可以以不同的格式存储，其中 Market Exchange Format (MEX) 和 Hierarchical Data Format (HDF5) 是两种不同的格式。</p><p data-tool="mdnice编辑器">其中Market Exchange Format (MEX)格式的表达量矩阵有3个文件，如下所示：</p><pre data-tool="mdnice编辑器"><span></span><code>$ <span>cd</span> /home/jdoe/runs/sample345/outs<br>$ tree filtered_feature_bc_matrix<br><br>filtered_feature_bc_matrix<br>├── barcodes.tsv.gz<br>├── features.tsv.gz<br>└── matrix.mtx.gz<br><br>0 directories, 3 files<br></code></pre><p data-tool="mdnice编辑器">而且这3个文件必须是有固定的格式以及固定好的文件名字，不能修改！！！因为我们读取它的时候只需要文件夹的名字，文件夹里面的3个文件是一定要固定的！分别存储 列名（细胞barcode），行名（基因名字），表达量矩阵（稀疏矩阵格式）。</p><p data-tool="mdnice编辑器">在R或者Python编程语言里面的，这3个文件都是可以分开独立读取的。使用 Python 的 <code>scipy</code> 库或 R 的 <code>Matrix</code> 和 <code>data.table</code> 等库来分别读取这三个单细胞转录组文件。</p><p data-tool="mdnice编辑器">在Python里面是 <code>csv</code>, <code>os</code>, <code>gzip</code>, and <code>scipy.io</code> modules ，下面是 Python 代码示例：</p><pre data-tool="mdnice编辑器"><span></span><code>import scipy.io<br>import pandas as pd<br><br><span># 读取 matrix.mtx.gz 文件</span><br>matrix = scipy.io.mmread(<span>'matrix.mtx.gz'</span>)<br><br><span># 读取 barcodes.tsv.gz 文件</span><br>barcodes_df = pd.read_csv(<span>'barcodes.tsv.gz'</span>, header=None, names=[<span>'barcodes'</span>])<br><br><span># 读取 features.tsv.gz 文件</span><br>features_df = pd.read_csv(<span>'features.tsv.gz'</span>, header=None, names=[<span>'features'</span>])<br><br><span># 打印示例数据</span><br>print(<span>"Matrix Shape:"</span>, matrix.shape)<br>print(<span>"Barcodes DataFrame:"</span>)<br>print(barcodes_df.head())<br>print(<span>"Features DataFrame:"</span>)<br>print(features_df.head())<br></code></pre><p data-tool="mdnice编辑器">下面是R 代码示例：</p><pre data-tool="mdnice编辑器"><span></span><code><span>library</span>(Matrix)<br><span>library</span>(data.table)<br><br><span># 读取 matrix.mtx.gz 文件</span><br>matrix &lt;- readMM(<span>"matrix.mtx.gz"</span>)<br><br><span># 读取 barcodes.tsv.gz 文件</span><br>barcodes_df &lt;- fread(<span>"barcodes.tsv.gz"</span>, header = <span>FALSE</span>, col.names = c(<span>"barcodes"</span>))<br><br><span># 读取 features.tsv.gz 文件</span><br>features_df &lt;- fread(<span>"features.tsv.gz"</span>, header = <span>FALSE</span>, col.names = c(<span>"features"</span>))<br><br><span># 打印示例数据</span><br>cat(<span>"Matrix Dimensions:"</span>, dim(matrix), <span>"\n"</span>)<br>cat(<span>"Barcodes DataFrame:\n"</span>)<br>head(barcodes_df)<br>cat(<span>"Features DataFrame:\n"</span>)<br>head(features_df)<br></code></pre><p data-tool="mdnice编辑器">这些代码将帮助你分别读取单细胞转录组的矩阵、条形码和特征文件。请确保你的 Python 环境中已经安装了 <code>scipy</code> 和 <code>pandas</code> 库，而 R 环境中已经安装了 <code>Matrix</code> 和 <code>data.table</code> 包。如果你熟悉这两个编程语言，安装模块和包对你来说应该是手到擒来的事情。</p><p data-tool="mdnice编辑器">而Hierarchical Data Format (HDF5) 格式的单细胞表达量矩阵就更简单了，简简单单的h5后缀的文件就存储上面的3个文件的全部信息，所以大概率是不需要大家分开读取上面的3个文件的，因为可以一次性读取。</p><h3
