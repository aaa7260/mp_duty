data-tool="mdnice编辑器"><span></span><span>用于基因注释的数据库：</span><span></span></h3><p data-tool="mdnice编辑器"><strong>refGene</strong></p><ul data-tool="mdnice编辑器"><li><section><strong>数据来源</strong>：数据库来自美国国家生物技术信息中心（NCBI）的RefSeq项目（https://www.ncbi.nlm.nih.gov/refseq/）</section></li><li><section><strong>内容</strong>：一组全面、完整、非冗余、注释良好的参考序列，包括基因的位置、转录本、编码区（CDS）的详细信息，以及每个转录本的外显子边界和长度。</section></li><li><section><strong>用途</strong>：用于基因级别的变异注释，帮助研究者判断变异落在基因的哪个区域（如外显子、内含子、UTR），以及变异对基因表达和功能的潜在影响。</section></li></ul><p data-tool="mdnice编辑器"><strong>refGeneWithVer</strong></p><ul data-tool="mdnice编辑器"><li><section><strong>数据来源</strong>：与refGene相同，也是来自NCBI的RefSeq项目。</section></li><li><section><strong>内容</strong>：除了包含refGene的所有信息外，还包括转录本版本号。</section></li><li><section><strong>用途</strong>：这个数据库的主要用途与refGene类似，但添加了转录本版本号，有助于精确跟踪和引用特定的转录本。</section></li></ul><p data-tool="mdnice编辑器"><strong>knownGene</strong></p><ul data-tool="mdnice编辑器"><li><section><strong>数据来源</strong>：来自UCSC的已知的基因（https://genome.ucsc.edu/）</section></li><li><section><strong>内容</strong>：包括基因的名称、位置、转录本和外显子信息，主要基于UCSC的整合数据。</section></li><li><section><strong>用途</strong>：用于基因和转录本级别的注释，有助于了解变异在基因表达水平上的潜在影响。</section></li></ul><p data-tool="mdnice编辑器"><strong>ensGene</strong></p><ul data-tool="mdnice编辑器"><li><section><strong>数据来源</strong>：基于Gencode注释的最新转录本信息（https://www.gencodegenes.org/）</section></li><li><section><strong>内容</strong>：包括由Gencode提供的基因名称、位置、转录本和外显子信息。</section></li><li><section><strong>用途</strong>：用于基因级别的变异注释，使研究者可以快速确定一个给定的遗传变异是否位于某个转录本的编码区、UTR区、内含子或其他调节区域。</section></li><li><section>ensGene40 和 ensGene41 ：数字特指Gencode的某个特定注释版本，使研究人员可以根据特定版本的注释来分析遗传变异</section></li></ul><h3 data-tool="mdnice编辑器"><span></span><span>基于区域的注释：</span><span></span></h3><p data-tool="mdnice编辑器"><strong>cytoBand</strong>：</p><ul data-tool="mdnice编辑器"><li><section><strong>来源</strong>：这个数据库来源于UCSC Genome Browser （https://genome.ucsc.edu/）</section></li><li><section><strong>内容</strong>：它包含染色体的带型信息，这些带型基于染色体的染色质结构差异，可以帮助定位染色体上的特定区域。</section></li><li><section><strong>用途</strong>：主要用于标记变异发生的染色体区域，有助于可视化和区域定位。</section></li></ul><h3 data-tool="mdnice编辑器"><span></span><span>基于过滤的注释</span><span></span></h3><h5 data-tool="mdnice编辑器"><span></span>基因变异预测和功能性数据库<span></span></h5><p data-tool="mdnice编辑器"><strong>dbNSFP</strong> 系列 (<code>dbnsfp30a</code>, <code>dbnsfp31a</code>, <code>dbnsfp33a</code>, <code>dbnsfp35a</code>, <code>dbnsfp35c</code>, <code>dbnsfp41a</code>, <code>dbnsfp41c</code>, <code>dbnsfp42a</code>, <code>dbnsfp42c</code>)</p><ul data-tool="mdnice编辑器"><li><section><strong>数据来源</strong>: 人类非同义和剪接位点 SNV 功能预测和注释的一站式数据库。综合了多个数据库，包括SIFT, PolyPhen-2, MutationTaster等预测工具的结果（http://database.liulab.science/dbNSFP）</section></li><li><section><strong>内容</strong>: 包括非同义突变的功能预测分数、保守性得分、基因组注释、蛋白质结构预测等。</section></li><li><section><strong>用途</strong>: 用于预测非同义单核苷酸变异对蛋白质功能的影响。</section></li></ul><p data-tool="mdnice编辑器"><strong>revel</strong></p><ul data-tool="mdnice编辑器"><li><section><strong>数据来源</strong>: 多种预测工具的综合评分（https://sites.google.com/site/revelgenomics/）</section></li><li><section><strong>内容</strong>: 针对人类编码非同义变异的集成风险评分。</section></li><li><section><strong>用途</strong>: 用于预测变异对蛋白功能的可能影响。</section></li></ul><h5 data-tool="mdnice编辑器"><span></span>常见变异数据库<span></span></h5><p data-tool="mdnice编辑器"><strong>avsnp</strong> 系列 (<code>avsnp142</code>, <code>avsnp144</code>, <code>avsnp147</code>, <code>avsnp150</code>)</p><ul data-tool="mdnice编辑器"><li><section><strong>数据来源</strong>: 基于NCBI的dbSNP数据库的不同版本（https://www.ncbi.nlm.nih.gov/snp/）</section></li><li><section><strong>内容</strong>: 包含广泛的单核苷酸多态性（SNP）及其rs编号。</section></li><li><section><strong>用途</strong>: 用于提供变异的常见性信息，帮助区分罕见变异和常见多态性。</section></li></ul><p data-tool="mdnice编辑器"><strong>1000 Genomes</strong> (<code>1000g2014oct</code>, <code>1000g2015aug</code>)</p><ul data-tool="mdnice编辑器"><li><section><strong>数据来源</strong>: 1000 Genomes Project （https://www.internationalgenome.org/）</section></li><li><section><strong>内容</strong>: 全球各种族群的单核苷酸多态性（SNPs）和小的插入删除（indels）。</section></li><li><section><strong>用途</strong>: 用于分析变异的群体频率，对遗传多样性研究非常重要。</section></li></ul><p data-tool="mdnice编辑器"><strong>GnomAD</strong> (<code>gnomad</code>, <code>gnomad211</code>, <code>gnomad30</code>, <code>gnomad312</code>)</p><ul data-tool="mdnice编辑器"><li><section><strong>数据来源</strong>:  Genome Aggregation Database — Broad Institute（https://gnomad.broadinstitute.org/）</section></li><li><section><strong>内容</strong>: 大规模全基因组和外显子测序数据，涵盖多种族群的频率数据。</section></li><li><section><strong>用途</strong>: 提供人群中变异的频率信息，对遗传研究和疾病关联分析很有帮助。</section></li></ul><p data-tool="mdnice编辑器"><strong>ExAC</strong> (<code>exac03</code>, <code>exac03nonpsych</code>, <code>exac03nontcga</code>)</p><ul data-tool="mdnice编辑器"><li><section><strong>数据来源</strong>: Exome Aggregation Consortium，收集了多个研究项目的外显子测序数据 （已迁移至 gnomAD : https://gnomad.broadinstitute.org/downloads#exac ）</section></li><li><section><strong>内容</strong>: 大规模外显子测序数据的变异频率。</section></li><li><section><strong>用途</strong>: 用于评估变异在较大人群中的频率，特别是在健康人群中。有助于判断变异的罕见程度及其可能的疾病相关性。</section></li></ul><h5 公共和私人测序项目的集合。（https://db.systemsbiology.net/kaviar/）</section></li><li><section><strong>内容</strong>: 全球人群中已知变异的集合。</section></li><li><section><strong>用途</strong>: 用于评估变异在人群中的普遍性。</section></li></ul><p
