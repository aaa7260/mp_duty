早期胃癌（GC）治愈率较高，但大多数患者发现时已经是晚期或转移期，预后相对较差。目前，针对PD-1和CTLA-4的抗体免疫治疗，已经引起了多种癌症（如黑色素瘤）的治疗模式的转变，但其在GC中的应答率相对较低。许多先前的研究表明，肿瘤间异质性和细胞组成的个体差异与生存率密切相关，强调了一种潜在需求，即剖析肿瘤微环境（TME）的复杂和动态生物学特征以干预GC。最近，单细胞RNA测序（scRNA-seq）目前已成功用于破译GC的微环境。</p><p>   </p><p>      在这篇文章中，作者应用scRNA-seq来绘制来自10名GC患者的肿瘤、癌旁正常组织和匹配有外周血中的免疫、基质和上皮细胞的转录图谱，并结合T细胞受体（TCR/BCR）全谱分析，希望能有助于胃癌TME和生物学特征的研究。</p><p><br></p></section><section><svg viewbox="0 0 1
