data-brushtype="text"><span>基于标记基因的评分方法</span></section></section></section><section><span><br></span></section><section><strong><span>（1）MCPcounter</span></strong><br></section><p><span>一种基于标记基因定量</span><span>肿瘤浸润免疫细胞（CD3 + T细胞，CD8 + T细胞，细胞毒性淋巴细胞，NK细胞，B淋巴细胞，单核细胞来源的细胞(单核细胞系)，髓样树突状细胞，中性粒细胞）、成纤维细胞和上皮细胞</span><span>的方法。对于每个细胞类型和样本，丰度得分为细胞类型特异性基因表达值的几何平均值，独立地对每个样本进行计算的。由于分数是用任意单位表示的，它们不能直接解释为细胞分数，也不能在细胞类型之间进行比较。进行定量验证时，估计分数与真实细胞分数之间具有很高的相关性，证明了MCP-counter用于</span><span>样本间比较</span><span>的价值。MCP-counter已应用于对32个非血液学肿瘤的19,000多个样本中的免疫细胞和非免疫细胞进行量化。</span></p><p><span><br></span></p><section><strong><span>（2）TIminer</span></strong><span></span></section><p><span>TIminer是一个用户友好的计算框架，用于不同的肿瘤免疫基因组分析，包括(i)来自NGS数据的</span><span>人白细胞抗原HLAs基因分型</span><span>；(ii)使用突变数据和HLA类型</span><span>预测肿瘤新抗原</span><span>；(iii)来自大量RNA-seq数据来分析定量肿瘤浸润免疫细胞；(iv)通过表达数据量化肿瘤的免疫原性。</span></p><p><span><br></span></p><section><strong><span>（3）xCell</span></strong><span></span></section><p><span>xCell是一种基于ssGSEA的方法，能够计算</span><span>64种免疫细胞的丰度分数，包括适应性和先天免疫细胞、造血祖细胞、上皮细胞和细胞外基质细胞</span><span>。基于FANTOM5，ENCODE，Blueprint，Immune Response In Silico (IRIS)，Human Primary Cell Atlas (HPCA)和Novershtern等6个研究的489个基因集。对于每种细胞类型，计算xCell丰度分数的主要步骤有四个：(i)利用R包GSVA对489个基因集单独进行ssGSEA；(ii)对属于一种细胞类型的所有基因集的ES进行平均；(iii)平台特异性ES转化为丰度分数；(iv)使用与流式细胞术数据分析类似的spillover方法纠正密切相关的细胞类型之间的关系。</span></p><section><span></span></section><section><span><br></span></section><section><section><section><span><strong>02</strong></span></section><section data-brushtype="text"><span><span>使用表达特征对细胞混合物进行反</span>褶<span>积</span></span></section></section></section><section><br></section><section><span><strong>（1）DeconRNASeq</strong></span></section><p><span>Gong和Szustakowski将约束回归模型应用于RNA-seq数据分析，并将其实现在R包DeconRNASeq中。</span><span>混合五种人体组织(大脑、骨骼肌、肺、肝和心脏)的RNA-seq数据</span><span>，通过合并组织类型特异性基因来估计组织或细胞类型的比例。并利用Illumina’s human Body Map 2.0中RNA-seq数据构建的特征矩阵，对算法进行了仿真验证。虽然还没有开发出新的免疫特征，但</span><span>该工具原则上可以应用于任何特征矩阵</span><span>。</span></p><p><span><br></span></p><section><span><strong>（2）CIBERSORT</strong></span></section><p><span>CIBERSORT算法利用微阵列数据构建特征矩阵，描述</span><span>22种免疫细胞表型的表达特征，包括不同的细胞类型和功能状态的免疫细胞</span><span>。CIBERSORT使用ν-SVR评估细胞分数。CIBERSORT分别在9个免疫细胞亚群和3个免疫细胞亚群的同时反褶积方面具有较高的准确性，在4种恶性免疫细胞的模拟混合物上测试，也证明了对不同程度的噪音和未知的肿瘤具有鲁棒性。这个工具在我们的公众号中有过介绍[<a target="_blank" T细胞、CD8 T细胞、中性粒细胞、巨噬细胞和树突状细胞）的丰度</span><span>。从RNA-seq或微阵列数据中提取的癌症表达矩阵与免疫细胞表达矩阵合并，并用Combat进行归一化，以消除批量效应。通过从免疫细胞标记中选择与肿瘤纯度负相关的基因，为每种癌症类型分别识别特征基因。最后，对于每种癌症类型，考虑到所选的免疫细胞标记，从标准化的免疫细胞配置文件构建特征矩阵。TIMER使用线性最小二乘回归方法执行反褶积，并强制所有负估计为零。用越来越小的一组T细胞标记重复估计几次，以减少CD8 +和CD4 + T细胞比例之间的相关性。与CIBERSORT不同的是，最终的估计并没有被规范化，因此不能直接解释为细胞组分，也不能在不同的免疫细胞类型和数据集之间进行比较。</span></p><p><span><br></span></p><section><strong><span>（4）EPIC</span></strong><span></span></section><p><span>Estimate (EPIC)，用来估计免疫细胞和癌细胞比例。EPIC使用约束最小二乘回归明确地将非负性约束引入反褶积问题，并要求每个样本中所有细胞组分的总和不超过一个。EPIC克服了以往从大量肿瘤基因表达数据预测癌症和免疫细胞或其他非恶性细胞类型的方法的几个局限性，考虑了非特征性和可能高度可变的细胞类型，并在算法上得到了发展。能够</span><span>广泛应用于大多数实体肿瘤</span><span>，如黑色素瘤和结直肠样本验证的情况，但</span><span>不适合血液恶性肿瘤</span><span>，如白血病或淋巴瘤。</span></p><p><span><br></span></p><section><strong><span>（5）quanTIseq</span></strong><span></span></section><p><span>quanTIseq是</span><span>专门为RNA-seq数据开发</span><span>的反褶积工具，能够准确</span><span>定量未知肿瘤的含量，以及量化整体组织的免疫细胞组份</span><span>。基于约束最小二乘回归和一个新的特征矩阵（来自51个纯化或富集的免疫细胞类型的RNA-seq数据集）。quanTIseq实现了一个完整的反褶积流程来分析RNA-seq数据，能够避免混合物和特征矩阵之间的不一致性。</span></p><section><span></span></section><section><br></section><section><section><section><span><strong>03</strong></span></section><section
